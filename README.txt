== Description ==

Wordpress plugin thats integrate Constant Contact to Ultimate Member

== Installation ==

1. Download and activate Ultimate Member plugin (https://wordpress.org/plugins/ultimate-member/)
2. Download and activate Contact Form 7 Newsletter plugin (https://wordpress.org/plugins/contact-form-7-newsletter/)
3. Upload 'um-constant-contact' plugin folder to the '/wp-content/plugins/' directory
4. Activate the plugin through the 'Plugins' menu in WordPress
